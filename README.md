## アプローチ
　「意識」を意識した設計を行ってみた。
 　具体的には、侍と犬は下記の特徴をもつ「意識」を持って行動する。
  - やるべきこと（どこのセルに向かって何をするか）
  - 意識レベル（時間とともに減少)
  - 意識の見直し（状況により意識レベルを引き上げる）
  
  この設計をベースに必要な「意識」を追加/削除していった。
  
## 反省
 - いつもながら細部の検討不足
 - 犬により侍の完全ブロックができる場合があるが、事前回避ができるはずだった。
 - 犬の価値が高いので犬を自由にさせるロジックが無かった
